Ce inseamna de fapt HTML

HTML = HyperText Markup Language

Acesta are rolul de a ajuta utilizatorul sa codeze o pagina din WWW asemenea unui limaj de programare obisnuit ca PHP, C sau Python.

WWW = Wolrd Wide Web

WWW- ul strange foarte multe pagini de web si le pune la un loc, intr-o retea bine organizata numita internet.

Atentie! Pentru a face un site super profesional si smecher cum ar fi Wikipedia, Youtube sau chiar Google -motor de cautare- , e necesar un staff mai mare: HTML, CSS, JavaScript, PHP, SQL, JQuery, Bootstrap. Din elemetele precizate, html-ul este cel mai usor.

Structura unui program HTML

<!DOCTYPE html >
<html>
<head>
</head>
<body>
</body>
</html>

Prima lectie este aproape cea mai grea, dar nu pentru mult timp daca iti dai seama de logica programului (Asta daca nu ai un prof tampit, cum am avut eu la inceput). Structura de baza a programului html este cea din dreapta. Primul rand este optional deoarece el se foloseste pentru ca computerul sa recunoasca cea mai noua versiune a HTML-ului in anumite situatii. Limbajul paginilor web a ajuns la cea de-a cincea versiune a sa, care a adus schimbari majore si actualizari ca forme geometrice si margini rontujite si fisiere video sau audio.

Pot sa fac s-asa?

Cu siguranta ca nu poti fi mereu atent la detalii si atunci iti vei da seama ca pur si simplu nu ai chef sa scrii de fiecare data aceste etichete cu litere mici... Norocul tau este ca html nu este case sensitive, adica poti scrie tagurile fara sa fie obligatoriu cu litere mici. De exemplu codurile: <html>, <HTML>, <Html> sunt echivalente intre ele si toate deschid tag-ul pentru pagina html.
